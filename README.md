 FinDash – Personal Finance Manager

FinDash is a modern, responsive, and user-friendly Personal Finance Management Dashboard designed to help individuals efficiently manage their personal finances. The application allows users to track income and expenses, create budgets, monitor savings goals, analyze spending patterns through interactive charts, and maintain a complete transaction history.

Built using HTML, CSS, JavaScript, and Chart.js, FinDash provides a clean and intuitive interface with dark mode support and browser-based data storage using Local Storage.
